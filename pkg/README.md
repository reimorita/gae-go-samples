##Explanation for Sample Code

1. auth.go

	Sample for UserAPI.

2. datastore.go

	Sample for datastore API.
	This sample show how to get and put entity of datastore.

3. mail.go

	Sample for mail API.
	This sample show how to send email.

4. memcache.go

	Sample for memcache.
	This sample show how to put and get cache data by memcache.
